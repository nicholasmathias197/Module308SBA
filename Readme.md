# Student Grade Calculator

## What This Does
This program calculates student grades.

## Inputs Needed:
1. Course information
2. List of assignments
3. Student homework submissions

## What It Calculates:
- Checks if homework is late
- Applies late penalty (10% off)
- Figures out percentage score for each assignment
- Calculates overall average for each student

## Output:
Shows each student's:
- Student ID number
- Overall grade average
- Grades for each assignment

## Example:
Student 125:
- Average: 98.5%
- Homework 1: 94%
- Homework 2: 100%

## How to Use:
Run the program. It uses example data and shows the grades.
